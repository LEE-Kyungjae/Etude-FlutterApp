# mafiaGameTipApp 

마피아42라는 게임앱의 tip커뮤니티를 목적으로 제작되었다.
내부기능으로는 각종 게임내 계산요소들을 담은 계산페이지, 게임내 각종수치들과 이용자들의 tip을 담은 tip게시판페이지, 익명전체채팅&그룹채팅,카드및 엽서교환&길드원모집&게임인원모집등을 할수있는 장터페이지를 포함하고있다.
